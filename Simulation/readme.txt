Title: *_SolarPathPlotHorizonRF_*
- functions to plot the solar path in the horizon reference frame and its polar plot

-> Run the file "example" to visualize the results


*Contents*

1) approxSolarPath - algorithm to approximate the solar position (elevation and azimuth angles)

2) calcSolarPath - function to contain the algorithm to calculate solar position 
*change this function for increased accuracy

3) horizonRF3DPlot - function to plot the solar path in the horizon reference frame 
*uses parameters returned by polarPlotSolarPath

4) polarPlotSolarPath - function to produce the polar plot of the solar path
